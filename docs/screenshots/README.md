## How to capture

1. Run `npm run dev`
2. Open `http://localhost:5173` in a browser
3. Use a 1440×900 or 1280×800 window for consistent sizing
4. Capture with browser screenshot tool or OS screenshot
5. Save PNGs into this folder with the filenames above
